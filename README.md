## 项目说明

使用Python来调用`mmmojo.dll`与其`XPlugin`通信，从而可以实现一些自动化的功能。

## 功能说明

- 实现微信的OCR功能，可以识别图片中的文字。

## 使用说明

#### 使用方法

```shell
pip -r requirements.txt

python3 main.py
```

#### 依赖(必要条件)

1. 新版微信
2. Python3.x
